# QuantumConformalPrediction

This is a Python project that applies Quantum Conformal Prediction to quantum models in the form of parameterized quantum circuits.

### configurations
This is where configurations for both models and results are defined

### conformal_prediction
This is where conformal prediction is applied to data generated by the models, and also where the data is generated using either Qiskit AER or IBMQ NISQ machines. 

### distributions
This is where distributions are defined for data to be taken from both for model training and conformal prediction purposes (calibration data, etc).

### figures
This is where figures and results are generated.

### models
This is where the angle encodings and hardware efficient ansatz are defined for the density estimation, regression and classification experiments.

### saved
This is where data, including models, figures and job data, are saved.

### training
This is where the trainers for the models are defined for different tasks.

### utils
This includes several utilities that are helpful in other areas of the project.

### virtual_environments
This is where virtual environments are kept. The torchquantum virtual environment is used in training and models, and anywhere just requires the most up to date version of whatever is requested by Python when running it.
